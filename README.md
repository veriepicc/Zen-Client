# Zen Client v3

**Zen Client v3** — a Minecraft hack client written entirely in **modern C++**. Built with **C++20 modules (`.ixx`)** for a fully modular and maintainable architecture.

---

## Features

- **Modern C++ Client:** Uses `.ixx` modules for clean and maintainable code.  
- **Minimal Dependencies:** Only includes what is absolutely necessary, including:  
  - [LibHat](https://github.com/BasedInc/libhat) — for robust signature scanning  
  - [MinHook](https://github.com/TsudaKageyu/minhook) — for fast and reliable function hooking  
  - [EnTT](https://github.com/skypjack/entt) — where needed internally  
- **Performance-First:** Optimized for **low overhead** and **maximum speed**.  
- **Future-Proof:** Designed to be easily extensible while keeping the codebase clean.
